# MLB Doping Analysis

Our project is based on the research paper “A switching dynamic generalized linear model to detect abnormal performances in Major League Baseball.”  This research paper uses statistical methods to identify abnormal performance or home run production in Major League Baseball and detect which players’ performances were unnaturally inflated when steroids were commonly used in the league. It models player performance using count data such as home run totals. Rather than the original Lahman’s Baseball Database, we will focus on batting data from the CMU CMSAC SURE Linear Models Project Dataset, which is cleaned from the baseballr package generated by FanGraphs that contains Major League Baseball batting statistics from 2010-2019 suitable for regression and GLM analysis. We will model how player performance changes over time and investigate what factors might explain unusually high or low outcomes.
